CV Tailor CLI

Intelligently tailor your CV to any job profile in seconds.

What is it?

CV Tailor is a high-performance command-line tool built in Rust. It takes your master CV and a job description, then automatically filters and reorders your skills to match the job requirements perfectly.

Get Started

Clone the repository and run the tool directly with Cargo:

cargo run -- --cv CV-T/cv.txt --profile CV-T/profile.txt --out CV-T/tailored.txt

(Make sure you have Rust and Cargo installed)
